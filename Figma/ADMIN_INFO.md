# VibeWorks Admin Panel

## Admin Credentials for Testing

### Owner Account (Full Access)
- **Email:** owner@vibehub.com
- **Password:** Create an account with this email through signup
- **Permissions:**
  - View all products from all merch groups
  - Approve/reject products submitted by merch admins
  - View analytics for all sales
  - Add products (auto-approved)
  - Delete any product

### Mode XL Merch Admin
- **Email:** modexl@admin.com
- **Password:** Create an account with this email through signup
- **Permissions:**
  - View/manage only Mode XL products
  - Add new Mode XL products (requires owner approval)
  - View analytics for Mode XL sales only
  - Delete own products

### KALT Merch Admin
- **Email:** kalt@admin.com
- **Password:** Create an account with this email through signup
- **Permissions:**
  - View/manage only KALT products
  - Add new KALT products (requires owner approval)
  - View analytics for KALT sales only
  - Delete own products

## How to Access Admin Panel

1. Sign up with one of the admin emails above
2. After logging in, you'll see an "Admin" link in the navigation bar
3. Click "Admin" to access your admin dashboard

## Features

### Owner Dashboard
- **Analytics Overview:** Total revenue, total orders, product count
- **Top Products:** See best-selling items across all merch groups
- **Product Approval:** Review and approve/reject products from merch admins
- **Product Management:** Add, edit, or delete any product
- **Sales Tracking:** View recent orders and sales data

### Merch Admin Dashboard
- **Analytics Overview:** Revenue and orders for your merch group only
- **Top Products:** See your best-selling items
- **Product Management:** Add, edit, or delete your products
- **Pending Approvals:** See which of your products are awaiting owner approval
- **Sales Tracking:** View orders containing your products

## Product Workflow

1. **Merch Admin** adds a new product → Status: "pending"
2. **Owner** reviews and approves/rejects → Status: "approved" or "rejected"
3. Only "approved" products appear in the public shop
4. Merch admins can see all their products (pending, approved, rejected)
5. Owner can see all products from all groups

## Notes

- Products added by the owner are auto-approved
- Products added by merch admins require owner approval
- Only approved products are visible to customers in the shop
- Each merch admin can only see their own sales analytics
- Owner can see combined analytics across all groups
